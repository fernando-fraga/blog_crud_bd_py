Sobre o Blog

Bem-vindo à nossa página sobre! Aqui você pode aprender mais sobre o nosso projeto, as funcionalidades oferecidas e as tecnologias utilizadas.

O que é este blog?

Este blog foi criado como uma plataforma simples e intuitiva, onde os usuários podem compartilhar seus pensamentos, artigos e ideias sobre diversos temas. Nosso objetivo é oferecer um espaço aberto para discussões, aprendizado e troca de conhecimentos, proporcionando uma experiência de leitura envolvente e agradável.

Funcionalidades principais

O blog foi projetado para ser fácil de usar, com as seguintes funcionalidades:

  • 💬 **Criação de posts** com título e conteúdo de forma simples.
  • ✏️ **Edição e exclusão de posts** criados pelo usuário.
  • 👀 **Visualização dos posts** de outros usuários para troca de ideias.

Sobre os Desenvolvedores

Este blog foi desenvolvido como parte de um projeto acadêmico da matéria de Tópicos Especiais. O objetivo principal foi criar um sistema funcional de blog com foco no aprendizado prático.

  • Fernando Fraga - Desenvolvedor Backend | Responsável pelo desenvolvimento do sistema backend, criação das APIs e integração com o banco de dados.

Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas e poderosas, para garantir uma experiência eficiente e agradável:

  • 🐍 **Flask** (Python) - Framework para desenvolvimento web rápido e leve.
  • 📦 **SQLAlchemy** - ORM para facilitar a interação com o banco de dados.
  • 🎨 **Bootstrap** - Framework front-end para criar uma interface bonita e responsiva.
  • 🗃 **SQLite** - Banco de dados leve e fácil de configurar, ideal para projetos pequenos e médios.
